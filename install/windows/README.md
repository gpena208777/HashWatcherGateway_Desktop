# Windows Service Wrapper

This project uses [WinSW](https://github.com/winsw/winsw) to run the Python gateway as a Windows service.

1. Download `WinSW-x64.exe`.
2. Place it in this folder as `winsw-x64.exe`.
3. Run PowerShell as Administrator.
4. Execute:

```powershell
.\scripts\install_windows_service.ps1
```
